## Метрика Чепина

### Классификация переменных

| Переменная | Тип | Описание |
|---|---|---|
| e | P | входной параметр события |
| args | P | входной параметр `main` |
| CB | M | модифицируемая переменная |
| B | M | модифицируемая переменная |
| L | M | модифицируемая переменная |
| imgs | M | модифицируемая переменная |
| k | C | управляющая переменная цикла |
| selectedIndex | C | управляющая переменная условия |
| lbl | T | временная переменная |

### Подсчёт

| Параметр | Значение |
|---|---|
| P | 2 |
| M | 4 |
| C | 2 |
| T | 1 |

**Формула**

Q = P + 2M + 3C + 0.5T

**Результат**

Q = 2 + 2×4 + 3×2 + 0.5×1 = **16.5**

## Метрика Муса

Метрика Муса используется для оценки **надёжности программного обеспечения** и требует данных тестирования.

| Параметр | Описание |
|---|---|
| λ₀ | начальная интенсивность отказов |
| λ(t) | интенсивность отказов в момент времени |
| μ | количество найденных отказов |
| t | время тестирования |
| R(t) | вероятность безотказной работы |

⚠ Для данного кода метрика Муса **не вычисляется**, так как отсутствуют данные тестирования и статистика отказов.
