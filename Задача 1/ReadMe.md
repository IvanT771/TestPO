## Найденные ошибки в программе

| Строка | Описание ошибки |
|------|----------------|
| **13** | Возможные неверные имена файлов. |
| **22** | Лишняя точка с запятой `;`. |
| **24** | Цикл начинается с `1` вместо `0`. |
| **27** | Используется `imgs[0]`, который не был загружен. |
| **28** | Неправильная высота области изображения. |
| **48** | Условие должно допускать индекс `0`. |
